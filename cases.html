Cluster health check — undetected firmware drift Cluster health check — onopgemerkte firmware-drift

A 6-node Hyper-V cluster running production workloads had not had a structured review in three years. The client suspected storage performance issues but could not pinpoint a cause. Een 6-node Hyper-V-cluster met productiewerklasten had drie jaar geen gestructureerde review gehad. De klant vermoedde opslagprestatieproblemen maar kon de oorzaak niet achterhalen.

Firmware version mismatch across nodes identified — two hosts running a build with a known NVMe stability regression.Firmware-versieverschil tussen nodes vastgesteld — twee hosts draaiden een build met een bekende NVMe-stabiliteitsfout.
CSV ownership imbalance causing hidden I/O bottleneck documented and corrected.Eigenaarsonbalans van CSV veroorzaakte verborgen I/O-knelpunt — gedocumenteerd en gecorrigeerd.
Full remediation completed within the 36-hour block.Volledige herstelwerkzaamheden afgerond binnen het 36-uurblok.

Multisite cluster — pre-migration assessment Multisite cluster — pre-migratie assessment

A hospital group with two datacentre sites planned a Windows Server 2019 → 2022 rolling upgrade while maintaining 99.9% uptime for clinical systems. The internal team needed an independent second opinion on the upgrade sequence. Een ziekenhuisgroep met twee datacentrumlocaties plande een rolling upgrade van Windows Server 2019 naar 2022, met behoud van 99,9% uptime voor klinische systemen. Het interne team had behoefte aan een onafhankelijk second opinion over de upgradereeks.

Quorum configuration risk identified for the inter-site stretch scenario — corrected before upgrade began.Quorumconfiguratie-risico vastgesteld voor het intersite-stretchscenario — gecorrigeerd vóór aanvang upgrade.

LogisticsLogistiek 144 h

S2D / Azure Local readiness & migration S2D / Azure Local readiness & migratie

A logistics company running 40+ VMs on aging SAN-backed Hyper-V wanted to evaluate a move to Storage Spaces Direct ahead of a hardware refresh cycle. No internal S2D expertise existed. Een logistiek bedrijf met 40+ VM's op verouderd SAN-gebaseerd Hyper-V wilde een overstap naar Storage Spaces Direct evalueren vóór een hardware-verversingsronde. Interne S2D-expertise ontbrak.

Hardware compatibility matrix built against proposed HCL — three originally specified NIC models removed and replaced before purchase.Hardware-compatibiliteitsmatrix opgesteld aan de hand van de voorgestelde HCL — drie oorspronkelijk gespecificeerde NIC-modellen vervangen vóór inkoop.
Network redesign for RDMA/RoCE delivered; network team briefed and validated configuration.Netwerkontwerp voor RDMA/RoCE opgeleverd; netwerkteam gebrieft en configuratie gevalideerd.
